<h1>📧 Python Attachment Mailer</h1>

<p>
Python Attachment Mailer is a <b>Python-based email automation tool</b> that allows users to send emails with custom messages and file attachments using the SMTP protocol. 
The program supports sending <b>single attachments, multiple attachments, and automatically compressing multiple files into a ZIP archive before sending</b>.
</p>

<p>
This project demonstrates how to build a simple mail client using Python and includes secure email transmission with <b>TLS</b>, file handling, and email <b>MIME structure management</b>.
</p>

<h2>🚀 Key Features</h2>

<ul>
<li>Send emails using <b>SMTP protocol</b></li>
<li>Custom <b>subject and message</b></li>
<li>Attach <b>single or multiple files</b></li>
<li>Automatically <b>compress files into ZIP</b></li>
<li>Secure email sending using <b>TLS encryption</b></li>
<li>Simple <b>command-line interface</b></li>
</ul>

<h2>🛠 Technologies Used</h2>

<ul>
<li>Python 3</li>
<li>smtplib</li>
<li>email.mime</li>
<li>zipfile</li>
<li>SMTP Protocol</li>
</ul>
