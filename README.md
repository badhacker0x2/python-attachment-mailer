Python Attachment Mailer is a Python-based email automation tool that allows users to send emails with custom messages and file attachments using the SMTP protocol. The program supports sending single attachments, multiple attachments, and automatically compressing multiple files into a ZIP archive before sending.

This project demonstrates how to build a simple mail client using Python and includes secure email transmission with TLS, file handling, and email MIME structure management.

Key Features
Send emails using SMTP protocol

Custom subject and message

Attach single or multiple files

Automatically compress files into ZIP

Secure email sending using TLS encryption

Simple command-line interface

Technologies Used
Python 3

smtplib

email.mime

zipfile

SMTP Protocol
